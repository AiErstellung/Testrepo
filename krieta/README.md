# KRIETA

KRIETA defines the **layout and structure** of short-form videos
before content generation.

## Purpose
- Define canvas, zones, overlays and timing
- Content-agnostic
- GUI-editable
- AI- & renderer-ready

## Modules
- `schema/`  JSON Schema for validation
- `examples/`  Valid example layouts

## Usage
Validate KRIETA JSON using any JSON-Schema validator (draft-07).
